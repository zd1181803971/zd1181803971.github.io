如何搭建Hexo简单教程

发表于 2020-04-10 阅读次数： 387 1 分钟

##如何搭建Hexo

  1. 安装nods.js

  2. 进入cmd，npm外网太慢，用taobao
    npm install -g cnpm –registry=https://registry.npm.taobao.org

  3. 安装hexo
    cnpm install -g hexo-cli

  4. 初始化hexo
    hexo init

  5. 启动
    hexo s

  6. 创建新的博客

    hexo n

##将hexo发布到github

  1. 安装 git
    cnpm install –save hexo-deployer-git
  2. 修改_config.yml文件
    type: git
    repo: https://github.com/zd1181803971/zd1181803971.github.io.git
    branch: master
  3. 部署
    hexo d
  4. 重新生成 发布
    hexo g
    hexo s
    hexo d
